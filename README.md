# PeekPeek 官方网站

这是PeekPeek应用的官方下载页面，提供iOS和Android版本的下载链接。

## 项目结构

- `index.html` - 主页面
- `styles.css` - 样式表
- `script.js` - JavaScript脚本
- `images/` - 图片资源目录

## 功能

- 提供App Store下载链接
- 提供TestFlight测试版下载链接
- 提供Android APK下载链接
